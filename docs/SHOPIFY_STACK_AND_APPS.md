## What the stack is supposed to accomplish

### Flow / automation layer
Use Flow + FlowMail + Flow Companion for:
- onboarding automations
- post-purchase actions
- notifications
- workflow triggers

### Subscription layer
Use Appstle for:
- recurring plans
- tier control
- billing state

### Upload / delivery layer
Use Uploadly and Pendora for:
- file intake
- digital delivery
- asset / certificate handling

### Trust / legal-tech presentation layer
Use theme sections and structured content for:
- how it works
- plans
- legal strength explanation
- certificate trust visuals
- consultation / booking CTAs

## Standing storefront requirements

- no placeholder client data
- no fabricated order history
- no fake metrics
- use real integrations or clean empty states
- keep pages conversion-focused and easy to understand
